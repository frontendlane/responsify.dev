Responsify

Responsify generates CSS calc() values that use % or vw to smoothen the change in element's size on different resolutions.

Example

By default <div> elements expand to fit the width of their container.

div {
    width: auto; /* default */
}

Suppose the <div> element needs to be 30% of its parent on resolutions below and equal to 1000px.

@media (max-width: 1000px) {
    div {
        width: 30%;
    }
}

The @media query causes the <div> to "jump" at 1000px. To smoothen the transition we add another @media query.

@media (min-width: 1001px) and (max-width: 1200px) {
    div {
        width: calc(-4200px + 450%);
    }
}

Value for the CSS width property comes from a simple but annoying-to-calculate equation. Responsify simplifies and automates the calculation.
